# AppHealthyFood

Nombre de la Aplicación: Healthy Food
Nombre: Daniela Gómez López 

Explicación de aplicación: 
Es una app para poder visuzalizar recetas de comidas saludables la cual contiene hasta el momento Página de inicio, Login, Registro, menú de las recetas saludables.

1. Registro de usuario
En la pantalla principal aparecerá la opción “¿No tienes cuenta? Registrarse”.
Al hacer clic en Registrarse, se abre el formulario de registro.
El usuario debe completar los siguientes campos:
Nombre
Correo electrónico
Contraseña
Después de ingresar los datos, debe presionar Crear cuenta.
Una vez creada la cuenta, la aplicación redirige automáticamente a la página principal.

2. Inicio de sesión
En la pantalla de inicio también se encuentra la opción Login.
Al ingresar al login, se muestran dos opciones:
Entrar a la cuenta
Registrarse (para quienes aún no tienen una cuenta)
Para iniciar sesión, es obligatorio diligenciar:
Correo electrónico (Gmail)
Contraseña
Si alguno de estos campos está vacío, no será posible acceder.
Una vez se ingresan los datos correctamente, se presiona Entrar, y el usuario accede a la aplicación.

3. Vista principal de la aplicación
Después de iniciar sesión, el usuario es dirigido a la pantalla principal, donde puede ver el menú de recetas disponibles hasta el momento.
Al presionar el botón volver, la aplicación redirige a la página principal.
